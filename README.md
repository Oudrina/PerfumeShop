# 🌸 Perfume Store

A simple perfume e-commerce product page built with React and Vite, created as a frontend learning project.

## 📌 About

This is a static single-page product listing for a fictional perfume store. It was built from scratch as a hands-on way to learn React fundamentals — components, props, data rendering, and CSS styling — without using any UI libraries or backend.

## 🏃 Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/perfume-store.git
```

2. Navigate into the project folder

```bash
cd perfume-store
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open your browser at `http://localhost:5173`
